What is web development?

Web development is the coding which allow website functionality on what we set it do, while we are coding or programming. Basically, it can be deals on non-design aspect of building websites that including writing mark-up. It comes from different ranges, such as: creating plain content page to composite web-based application, business application and many more that you can do, if you are web designed.

The web development has three hierarchy that we follow:

  1. Client-side coding
  2. Server-side coding
  3. Database technology
